# 🚀 Projects Included

## 💬 Simple Messenger App

A real-time messaging platform built to explore the mechanics of web communication and data handling.

### Key Learning Objectives:

- **WebSockets (WS/WSS)**: Implemented full-duplex communication channels for instant message delivery.
- **HTTP Protocol**: Contrasting request-response cycles with persistent socket connections.
- **Data Handling**: Working with **Buffers** and **Strings** to manage binary data and message encoding across the wire.
- **Real-time State**: Managing active user sessions and message broadcasting logic.

### 🛠 Tech Highlights

- **Backend**: Node.js with `ws` library (or Socket.io)
- **Frontend**: React hooks for real-time UI updates
- **Concepts**: Event-driven architecture, binary data buffers, and low-latency networking.
